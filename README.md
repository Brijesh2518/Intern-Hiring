# Intern-Hiring
Intern-Hiring System


**Prerequisites:**  Node.js
Initialize a Node.js project: npm init -y
Install React:npm install react react-dom
Install development tools: npm install --save-dev vite @vitejs/plugin-react typescript @types/react @types/react-dom


1. Install dependencies:
   `npm install`
3. Run the app:
   `npm run dev`
